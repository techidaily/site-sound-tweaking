**Step 2:** You have to select a video having multiple audio tracks on “source selection” pane.

**Step 3:** File selection will generate an interface beneath. Select “audio tab” and expand the first dropdown which lists down all the audio tracks in a selected file. Select one of them and name the file and enter the output directory.
